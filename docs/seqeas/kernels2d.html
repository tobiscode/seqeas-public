  9def Glinedisp(x1, x2, x1midfp, x2midfp, halflen, theta, nu):
 10    """
 11    Compute the displacement Green's functions for a shearing and opening
 12    line crack in a halfspace. Adapted for Python from the ``LDdispHS``
 13    Matlab function from [davis17]_. References to "C&S" in the source code
 14    mean [crouchstarfield83]_.
 15
 16    Parameters
 17    ----------
 18    x1 : float, numpy.ndarray
 19        :math:`x_1` location of evaluation points [m]
 20    x2 : float, numpy.ndarray
 21        :math:`x_2` location of evaluation points [m]
 22    x1midfp : float, numpy.ndarray
 23        :math:`x_1` location of the midpoint of the fault patches [m]
 24    x2midfp : float, numpy.ndarray
 25        :math:`x_2` location of the midpoint of the fault patches [m]
 26    halflen : float, numpy.ndarray
 27        Half-length of each fault patch [m]
 28    theta : float, numpy.ndarray
 29        The angle [rad] between the fault patch orientation (not the fault
 30        normal) and the :math:`x_1` axis in the mathematically negative sense,
 31        i.e. positive clockwise.
 32    nu : float
 33        Poisson's ratio [-]
 34
 35    Returns
 36    -------
 37    G : numpy.ndarray
 38        Displacement kernel for all patches and all evaluation points [m].
 39        The first half of rows corresponds to the :math:`xx` component,
 40        the second one to :math:`yy`.
 41
 42    Notes
 43    -----
 44
 45    To compute the displacement at the evaluation points given the shear and normal
 46    displacements :math:`D_s` and :math:`D_n` in each patch [m], with positive
 47    values meaning left-lateral shearing and crack opening, respectively,
 48    perform a matrix computation of the Green's functions with the displacements.
 49
 50    Example
 51    -------
 52
 53    Calculate the stress kernel for three observation points and two fault
 54    patches:
 55
 56    >>> import numpy as np
 57    >>> x1, x2 = np.meshgrid([-1, 0, 1], [0])
 58    >>> x1, x2 = x1.ravel(), x2.ravel()
 59    >>> x1midfp, x2midfp = [0.5, 1.5], [-0.5, -1.5]
 60    >>> halflen, theta, nu = 1/np.sqrt(2), np.pi/4, 0.25
 61    >>> G = Glinedisp(x1, x2, x1midfp, x2midfp, halflen, theta, nu)
 62
 63    Calculate the displacement given left-lateral shearing and crack opening in
 64    both patches:
 65
 66    >>> Ds, Dn = [1, 1], [0.1, 0.1]
 67    >>> D = G @ np.concatenate([Ds, Dn])
 68    >>> print(D.round(3))
 69    [ 0.289    nan -0.133 -0.094    nan  0.441]
 70
 71    (The ``nan`` values are where the fault patch breaks the surface.)
 72
 73    References
 74    ----------
 75
 76    .. [davis17] Davis, T. (2017).
 77       *A new open source boundary element code and its application to geological*
 78       *deformation: Exploring stress concentrations around voids and the effects*
 79       *of 3D frictional distributions on fault surfaces* (M.Sc thesis. Aberdeen University)
 80
 81    .. [crouchstarfield83] Crouch, S. L., & Starfield, A. M. (1983).
 82       *Boundary element methods in solid mechanics: with applications in rock*
 83       *mechanics and geological engineering*. Allen & Unwin.
 84    """
 85    # format shapes
 86    x1 = np.asarray(x1).reshape(-1, 1)
 87    x2 = np.asarray(x2).reshape(-1, 1)
 88    x1midfp = np.asarray(x1midfp).reshape(1, -1)
 89    x2midfp = np.asarray(x2midfp).reshape(1, -1)
 90    halflen = np.asarray(halflen).reshape(1, -1)
 91    beta = np.asarray(-theta).reshape(1, -1)
 92
 93    # input check
 94    if np.any(x2 > 0) or np.any(x2midfp > 0):
 95        raise ValueError("x_2 locations need to be zero or less in a halfspace.")
 96
 97    # Define material constant used in calculating influence coefficients.
 98    con = 1 / (4 * np.pi * (1 - float(nu)))
 99    sb, cb = np.sin(beta), np.cos(beta)
100    s2b, c2b = np.sin(2 * beta), np.cos(2 * beta)
101    s3b, c3b = np.sin(3 * beta), np.cos(3 * beta)
102
103    # Define array of local coordinates for the observation grid relative to
104    # the midpoint and orientation of the ith element.
105    # Refer to (Figure 5.6, C&S, p. 91) and eqs. 4.5.1 of C&S, p. 57.
106    XB = (x1 - x1midfp) * cb + (x2 - x2midfp) * sb
107    YB = -(x1 - x1midfp) * sb + (x2 - x2midfp) * cb
108
109    # Coordinates of the image dislocation (equation 7.4.6 C&S)
110    XBi = (x1 - x1midfp) * cb - (x2 + x2midfp) * sb
111    YBi = (x1 - x1midfp) * sb + (x2 + x2midfp) * cb
112
113    # Fix roundoff errors in Ybi and Yb from trig function problems
114    YB[np.abs(YB) < 1e-10] = 0
115    YBi[np.abs(YBi) < 1e-10] = 0
116
117    # Calculate derivatives of the function f(x,y), eq. 5.2.5 of C&S, p. 81.
118    # which are used to calculate the displacement and stress components.
119    # It is understood that X and Y refer to XB and YB.
120    # First abbreviate repeated terms in the derivatives of f(x,y):
121    Y2 = YB**2
122    XMa, XPa = XB - halflen, XB + halflen
123    XMa2, XPa2 = XMa**2, XPa**2
124    R1S = XMa2 + Y2
125    R2S = XPa2 + Y2
126
127    # Same thing for the image dislocation
128    Y2i = YBi**2
129    XMai, XPai = XBi - halflen, XBi + halflen
130    XMa2i, XPa2i = XMai**2, XPai**2
131    R1Si = XMa2i + Y2i
132    R2Si = XPa2i + Y2i
133
134    # don't solve for locations where R1S(i) or R2S(i) are zero
135    skip_ix = (R1S == 0) | (R2S == 0) | (R1Si == 0) | (R2Si == 0)
136    R1S[skip_ix] = np.nan
137    R2S[skip_ix] = np.nan
138    R1Si[skip_ix] = np.nan
139    R2Si[skip_ix] = np.nan
140
141    R1S2i = R1Si**2
142    R2S2i = R2Si**2
143
144    # The following derivatives are eqs. 4.5.5a thru d of C&S, p. 58.
145    FF2 = con * (np.log(np.sqrt(R1S)) - np.log(np.sqrt(R2S)))
146
147    # get FF3 directly from engle differences without subindexing
148    FF3 = -con * np.angle(
149        np.exp(1j * np.arctan2(YB, XMa)) * np.exp(-1j * np.arctan2(YB, XPa))
150    )
151
152    # get the next derivatives
153    FF4 = con * (YB / R1S - YB / R2S)
154    FF5 = con * (XMa / R1S - XPa / R2S)
155
156    # Calculate intermediate functions Fni for the image dislocation
157    FF2i = con * (np.log(np.sqrt(R1Si)) - np.log(np.sqrt(R2Si)))  # Equations 4.5.5 C&S
158    FF3i = -con * np.angle(
159        np.exp(1j * np.arctan2(YBi, XMai)) * np.exp(-1j * np.arctan2(YBi, XPai))
160    )
161    FF4i = con * (YBi / R1Si - YBi / R2Si)
162    FF5i = con * (XMai / R1Si - XPai / R2Si)
163
164    # The halfspace examples of eqs. 5.5.3a and b of C&S, p. 91.
165    # See Appendix A of: Martel, S.J. and Langley, J.S., 2006. Propagation of
166    # normal faults to the surface in basalt, Koae fault system, Hawaii.
167    # Journal of Structural Geology, 28(12), pp.2123-2143.
168    FF6i = con * ((XMa2i - Y2i) / R1S2i - (XPa2i - Y2i) / R2S2i)
169    FF7i = 2 * con * YBi * (XMai / R1S2i - XPai / R2S2i)
170
171    # Define material constants used in calculating displacements.
172    pr1 = 1 - 2 * nu
173    pr2 = 2 * (1 - nu)
174    pr3 = 3 - 4 * nu
175
176    # Calculate the displacement components using eqs. 5.5.4 of C&S, p. 91.
177    Gxx_s = -pr1 * sb * FF2 + pr2 * cb * FF3 + YB * (sb * FF4 - cb * FF5)
178    Gxy_n = -pr1 * cb * FF2 - pr2 * sb * FF3 - YB * (cb * FF4 + sb * FF5)
179    Gyx_s = +pr1 * cb * FF2 + pr2 * sb * FF3 - YB * (cb * FF4 + sb * FF5)
180    Gyy_n = -pr1 * sb * FF2 + pr2 * cb * FF3 - YB * (sb * FF4 - cb * FF5)
181
182    # See equations 7.4.8 and 7.4.9 in Crouch and Starfield
183    # Calculate IMAGE AND SUPPLEMENTAL DISPLACEMENT components due to unit SHEAR
184    # displacement discontinuity
185    Gxxi_s = (
186        pr1 * sb * FF2i
187        - pr2 * cb * FF3i
188        + (pr3 * (x2 * s2b - YB * sb) + 2 * x2 * s2b) * FF4i
189        + (pr3 * (x2 * c2b - YB * cb) - x2 * (1 - 2 * c2b)) * FF5i
190        + 2 * x2 * (x2 * s3b - YB * s2b) * FF6i
191        - 2 * x2 * (x2 * c3b - YB * c2b) * FF7i
192    )
193    Gyxi_s = (
194        -pr1 * cb * FF2i
195        - pr2 * sb * FF3i
196        - (pr3 * (x2 * c2b - YB * cb) + x2 * (1 - 2 * c2b)) * FF4i
197        + (pr3 * (x2 * s2b - YB * sb) - 2 * x2 * s2b) * FF5i
198        + 2 * x2 * (x2 * c3b - YB * c2b) * FF6i
199        + 2 * x2 * (x2 * s3b - YB * s2b) * FF7i
200    )
201
202    # Calculate IMAGE AND SUPPLEMENTAL DISPLACEMENT components due to unit NORMAL
203    # displacement discontinuity
204    Gxyi_n = (
205        pr1 * cb * FF2i
206        + pr2 * sb * FF3i
207        - (pr3 * (x2 * c2b - YB * cb) - x2) * FF4i
208        + pr3 * (x2 * s2b - YB * sb) * FF5i
209        - 2 * x2 * (x2 * c3b - YB * c2b) * FF6i
210        - 2 * x2 * (x2 * s3b - YB * s2b) * FF7i
211    )
212    Gyyi_n = (
213        pr1 * sb * FF2i
214        - pr2 * cb * FF3i
215        - pr3 * (x2 * s2b - YB * sb) * FF4i
216        - (pr3 * (x2 * c2b - YB * cb) + x2) * FF5i
217        + 2 * x2 * (x2 * s3b - YB * s2b) * FF6i
218        - 2 * x2 * (x2 * c3b - YB * c2b) * FF7i
219    )
220
221    Gxx = Gxx_s + Gxxi_s
222    Gxy = -(Gxy_n + Gxyi_n)
223    Gyx = Gyx_s + Gyxi_s
224    Gyy = -(Gyy_n + Gyyi_n)
225
226    # return stacked displacement kernel
227    Gx = np.hstack([Gxx, Gxy])
228    Gy = np.hstack([Gyx, Gyy])
229    G = np.vstack([Gx, Gy])
230    return G

233def Klinedisp(x1, x2, x1midfp, x2midfp, halflen, theta, nu, E):
234    """
235    Compute the stress kernel for a shearing and opening
236    line crack in a halfspace. Adapted for Python from the ``LDstressHS``
237    Matlab function from [davis17]_. References to "C&S" in the source code
238    mean [crouchstarfield83]_.
239
240    Parameters
241    ----------
242    x1 : float, numpy.ndarray
243        :math:`x_1` location of evaluation points [m]
244    x2 : float, numpy.ndarray
245        :math:`x_2` location of evaluation points [m]
246    x1midfp : float, numpy.ndarray
247        :math:`x_1` location of the midpoint of the fault patches [m]
248    x2midfp : float, numpy.ndarray
249        :math:`x_2` location of the midpoint of the fault patches [m]
250    halflen : float, numpy.ndarray
251        Half-length of each fault patch [m]
252    theta : float, numpy.ndarray
253        The angle [rad] between the fault patch orientation (not the fault
254        normal) and the :math:`x_1` axis in the mathematically negative sense,
255        i.e. positive clockwise.
256    nu : float
257        Poisson's ratio [-]
258    E : float
259        Young's modulus [Pa]
260
261    Returns
262    -------
263    K : numpy.ndarray
264        Stress kernel for all patches and all evaluation points [m].
265        The first third of rows corresponds to the :math:`xx` component,
266        the second to :math:`yy`, and the third to :math:`xy`.
267
268    Notes
269    -----
270
271    To compute the stress at the evaluation points given the shear and normal
272    displacements :math:`D_s` and :math:`D_n` in each patch [m], with positive
273    values meaning left-lateral shearing and crack opening, respectively,
274    perform a matrix computation of the stress kernel with the displacements.
275
276    Example
277    -------
278
279    Calculate the stress kernel for three observation points and two fault
280    patches:
281
282    >>> import numpy as np
283    >>> x1, x2 = np.meshgrid([-1, 0, 1], [0])
284    >>> x1, x2 = x1.ravel(), x2.ravel()
285    >>> x1midfp, x2midfp = [0.5, 1.5], [-0.5, -1.5]
286    >>> halflen, theta, nu, E = 1/np.sqrt(2), np.pi/4, 0.25, 1
287    >>> K = Klinedisp(x1, x2, x1midfp, x2midfp, halflen, theta, nu, E)
288
289    Calculate the stress given left-lateral shearing and crack opening in
290    both patches:
291
292    >>> Ds, Dn = [1, 1], [0.1, 0.1]
293    >>> S = K @ np.concatenate([Ds, Dn])
294    >>> print(S.round(3))
295    [ 0.084    nan  0.119 -0.       nan  0.     0.       nan  0.   ]
296
297    (The ``nan`` values are where the fault patch breaks the surface.)
298
299    """
300    # format shapes
301    x1 = np.asarray(x1).reshape(-1, 1)
302    x2 = np.asarray(x2).reshape(-1, 1)
303    x1midfp = np.asarray(x1midfp).reshape(1, -1)
304    x2midfp = np.asarray(x2midfp).reshape(1, -1)
305    halflen = np.asarray(halflen).reshape(1, -1)
306    beta = np.asarray(-theta).reshape(1, -1)
307
308    # input check
309    if np.any(x2 > 0) or np.any(x2midfp > 0):
310        raise ValueError("x_2 locations need to be zero or less in a halfspace.")
311
312    # The shear modulus, sm, is related to the prescribed elastic constants.
313    sm = E / (2 * (1 + float(nu)))
314    # Define material constant used in calculating influence coefficients.
315    con = 1 / (4 * np.pi * (1 - float(nu)))
316    cons = 2 * sm
317    sb, cb = np.sin(beta), np.cos(beta)
318    s2b, c2b = np.sin(2 * beta), np.cos(2 * beta)
319    s3b, c3b = np.sin(3 * beta), np.cos(3 * beta)
320    s4b, c4b = np.sin(4 * beta), np.cos(4 * beta)
321
322    # Define array of local coordinates for the observation grid relative to
323    # the midpoint and orientation of the ith element.
324    # Refer to (Figure 5.6, C&S, p. 91) and eqs. 4.5.1 of C&S, p. 57.
325    XB = (x1 - x1midfp) * cb + (x2 - x2midfp) * sb
326    YB = -(x1 - x1midfp) * sb + (x2 - x2midfp) * cb
327
328    # Coordinates of the image dislocation (equation 7.4.6 C&S)
329    XBi = (x1 - x1midfp) * cb - (x2 + x2midfp) * sb
330    YBi = (x1 - x1midfp) * sb + (x2 + x2midfp) * cb
331
332    # Fix roundoff errors in Ybi and Yb from trig function problems
333    YB[np.abs(YB) < 1e-10] = 0
334    YBi[np.abs(YBi) < 1e-10] = 0
335
336    # Calculate derivatives of the function f(x,y), eq. 5.2.5 of C&S, p. 81.
337    # which are used to calculate the displacement and stress components.
338    # It is understood that X and Y refer to XB and YB.
339    # First abbreviate repeated terms in the derivatives of f(x,y):
340    Y2 = YB**2
341    XMa, XPa = XB - halflen, XB + halflen
342    XMa2, XPa2 = XMa**2, XPa**2
343    R1S = XMa2 + Y2
344    R2S = XPa2 + Y2
345
346    # Same thing for the image dislocation
347    Y2i = YBi**2
348    XMai, XPai = XBi - halflen, XBi + halflen
349    XMa2i, XPa2i = XMai**2, XPai**2
350    R1Si = XMa2i + Y2i
351    R2Si = XPa2i + Y2i
352
353    # don't solve for locations where R1S(i) or R2S(i) are zero
354    skip_ix = (R1S == 0) | (R2S == 0) | (R1Si == 0) | (R2Si == 0)
355    R1S[skip_ix] = np.nan
356    R2S[skip_ix] = np.nan
357    R1Si[skip_ix] = np.nan
358    R2Si[skip_ix] = np.nan
359
360    R1S2, R2S2 = R1S**2, R2S**2
361    R1S2i, R2S2i = R1Si**2, R2Si**2
362
363    FF4 = con * (YB / R1S - YB / R2S)
364    FF5 = con * (XMa / R1S - XPa / R2S)
365
366    # The following derivatives are eqs. 5.5.3a and b of C&S, p. 91.
367    FF6 = con * ((XMa2 - Y2) / R1S2 - (XPa2 - Y2) / R2S2)
368    FF7 = 2 * con * YB * (XMa / R1S2 - XPa / R2S2)
369
370    FF4i = con * (YBi / R1Si - YBi / R2Si)
371    FF5i = con * (XMai / R1Si - XPai / R2Si)
372
373    # The halfspace examples of eqs. 5.5.3a and b of C&S, p. 91.
374    # See Appendix A of: Martel, S.J. and Langley, J.S., 2006. Propagation of
375    # normal faults to the surface in basalt, Koae fault system, Hawaii.
376    # Journal of Structural Geology, 28(12), pp.2123-2143.
377    FF6i = con * ((XMa2i - Y2i) / R1S2i - (XPa2i - Y2i) / R2S2i)
378    FF7i = 2 * con * YBi * (XMai / R1S2i - XPai / R2S2i)
379
380    # don't estimate at invalid points
381    HR1 = (halflen + XBi) ** 2 + YBi**2
382    HR2 = YBi**2 + (halflen - XBi) ** 2
383    skip_ix = (HR1 == 0) | (HR2 == 0)
384    HR1[skip_ix] = np.nan
385    HR2[skip_ix] = np.nan
386
387    # *Tim* I used MATLABs symbolic to find these not eq's A.3 and A.4 of Martel
388    # Used Eq.A.1 on variable FF7i (expanded).
389    FF8i = (
390        YBi
391        * (
392            1 / HR1**2
393            - 1 / HR2**2
394            + (2 * (halflen - XBi) * (2 * halflen - 2 * XBi)) / HR2**3
395            - (2 * (halflen + XBi) * (2 * halflen + 2 * XBi)) / HR1**3
396        )
397    ) / (2 * np.pi * (nu - 1))
398    FF9i = ((halflen - XBi) / HR2**2 + (halflen + XBi) / HR1**2) / (
399        2 * np.pi * (nu - 1)
400    ) - (
401        YBi
402        * ((4 * YBi * (halflen + XBi)) / HR1**3 + (4 * YBi * (halflen - XBi)) / HR2**3)
403    ) / (
404        2 * np.pi * (nu - 1)
405    )
406
407    # Calculate the stress components using eqs. 5.5.5 of C&S, p. 92.
408    Kxx_s = 2 * (cb * cb) * FF4 + s2b * FF5 + YB * (c2b * FF6 - s2b * FF7)
409    Kxx_n = -FF5 + YB * (s2b * FF6 + c2b * FF7)
410    Kyy_s = 2 * (sb * sb) * FF4 - s2b * FF5 - YB * (c2b * FF6 - s2b * FF7)
411    Kyy_n = -FF5 - YB * (s2b * FF6 + c2b * FF7)
412    Kxy_s = s2b * FF4 - c2b * FF5 + YB * (s2b * FF6 + c2b * FF7)
413    Kxy_n = -YB * (c2b * FF6 - s2b * FF7)
414
415    #  Calculate IMAGE AND SUPPLEMENTAL STRESS components due to unit SHEAR and
416    #  NORMAL displacement discontinuity
417    Kxxi_s = (
418        FF4i
419        - 3 * (c2b * FF4i - s2b * FF5i)
420        + (2 * x2 * (cb - 3 * c3b) + 3 * YB * c2b) * FF6i
421        + (2 * x2 * (sb - 3 * s3b) + 3 * YB * s2b) * FF7i
422        - 2 * x2 * (x2 * c4b - YB * c3b) * FF8i
423        - 2 * x2 * (x2 * s4b - YB * s3b) * FF9i
424    )
425
426    Kxxi_n = (
427        FF5i
428        + (2 * x2 * (sb - 2 * s3b) + 3 * YB * s2b) * FF6i
429        - (2 * x2 * (cb - 2 * c3b) + 3 * YB * c2b) * FF7i
430        - 2 * x2 * (x2 * s4b - YB * s3b) * FF8i
431        + 2 * x2 * (x2 * c4b - YB * c3b) * FF9i
432    )
433
434    Kyyi_s = (
435        FF4i
436        - (c2b * FF4i - s2b * FF5i)
437        - (4 * x2 * sb * s2b - YB * c2b) * FF6i
438        + (4 * x2 * sb * c2b + YB * s2b) * FF7i
439        + 2 * x2 * (x2 * c4b - YB * c3b) * FF8i
440        + 2 * x2 * (x2 * s4b - YB * s3b) * FF9i
441    )
442
443    Kyyi_n = (
444        FF5i
445        - (2 * x2 * sb - YB * s2b) * FF6i
446        + (2 * x2 * cb - YB * c2b) * FF7i
447        + 2 * x2 * (x2 * s4b - YB * s3b) * FF8i
448        - 2 * x2 * (x2 * c4b - YB * c3b) * FF9i
449    )
450
451    Kxyi_s = (
452        s2b * FF4i
453        + c2b * FF5i
454        + (2 * x2 * sb * (1 + 4 * c2b) - YB * s2b) * FF6i
455        + (2 * x2 * cb * (3 - 4 * c2b) + YB * c2b) * FF7i
456        + 2 * x2 * (x2 * s4b - YB * s3b) * FF8i
457        - 2 * x2 * (x2 * c4b - YB * c3b) * FF9i
458    )
459
460    Kxyi_n = (
461        (4 * x2 * sb * s2b + YB * c2b) * FF6i
462        - (4 * x2 * sb * c2b - YB * s2b) * FF7i
463        - 2 * x2 * (x2 * c4b - YB * c3b) * FF8i
464        - 2 * x2 * (x2 * s4b - YB * s3b) * FF9i
465    )
466
467    Kxx = np.hstack([Kxx_s + Kxxi_s, -Kxx_n - Kxxi_n])
468    Kyy = np.hstack([Kyy_s + Kyyi_s, -Kyy_n - Kyyi_n])
469    Kxy = np.hstack([Kxy_s + Kxyi_s, -Kxy_n - Kxyi_n])
470
471    # return stacked stress kernel
472    K = cons * np.vstack([Kxx, Kyy, Kxy])
473    return K

225    Gyy = -(Gyy_n + Gyyi_n)

228    Gx = np.hstack([Gxx, Gxy])
229    Gy = np.hstack([Gyx, Gyy])

430        - (2 * x2 * (cb - 2 * c3b) + 3 * YB * c2b) * FF7i
431        - 2 * x2 * (x2 * s4b - YB * s3b) * FF8i
432        + 2 * x2 * (x2 * c4b - YB * c3b) * FF9i
433    )
434
435    Kyyi_s = (
436        FF4i
437        - (c2b * FF4i - s2b * FF5i)
438        - (4 * x2 * sb * s2b - YB * c2b) * FF6i
439        + (4 * x2 * sb * c2b + YB * s2b) * FF7i
440        + 2 * x2 * (x2 * c4b - YB * c3b) * FF8i
441        + 2 * x2 * (x2 * s4b - YB * s3b) * FF9i
442    )
443
444    Kyyi_n = (
445        FF5i
446        - (2 * x2 * sb - YB * s2b) * FF6i
447        + (2 * x2 * cb - YB * c2b) * FF7i
448        + 2 * x2 * (x2 * s4b - YB * s3b) * FF8i
449        - 2 * x2 * (x2 * c4b - YB * c3b) * FF9i
450    )
451
452    Kxyi_s = (
453        s2b * FF4i
454        + c2b * FF5i
455        + (2 * x2 * sb * (1 + 4 * c2b) - YB * s2b) * FF6i
456        + (2 * x2 * cb * (3 - 4 * c2b) + YB * c2b) * FF7i
457        + 2 * x2 * (x2 * s4b - YB * s3b) * FF8i
458        - 2 * x2 * (x2 * c4b - YB * c3b) * FF9i
459    )
460
461    Kxyi_n = (
462        (4 * x2 * sb * s2b + YB * c2b) * FF6i
463        - (4 * x2 * sb * c2b - YB * s2b) * FF7i
464        - 2 * x2 * (x2 * c4b - YB * c3b) * FF8i
465        - 2 * x2 * (x2 * s4b - YB * s3b) * FF9i
466    )
467
468    Kxx = np.hstack([Kxx_s + Kxxi_s, -Kxx_n - Kxxi_n])
469    Kyy = np.hstack([Kyy_s + Kyyi_s, -Kyy_n - Kyyi_n])
470    Kxy = np.hstack([Kxy_s + Kxyi_s, -Kxy_n - Kxyi_n])
471
472    # return stacked stress kernel
473    K = cons * np.vstack([Kxx, Kyy, Kxy])
474    return K

Compute the stress kernel for a shearing and opening line crack in a halfspace. Adapted for Python from the LDstressHS Matlab function from [davis17]_. References to "C&S" in the source code mean [crouchstarfield83]_.

Parameters
  • x1 (float, numpy.ndarray): \( x_1 \) location of evaluation points [m]
  • x2 (float, numpy.ndarray): \( x_2 \) location of evaluation points [m]
  • x1midfp (float, numpy.ndarray): \( x_1 \) location of the midpoint of the fault patches [m]
  • x2midfp (float, numpy.ndarray): \( x_2 \) location of the midpoint of the fault patches [m]
  • halflen (float, numpy.ndarray): Half-length of each fault patch [m]
  • theta (float, numpy.ndarray): The angle [rad] between the fault patch orientation (not the fault normal) and the \( x_1 \) axis in the mathematically negative sense, i.e. positive clockwise.
  • nu (float): Poisson's ratio [-]
  • E (float): Young's modulus [Pa]
Returns
  • K (numpy.ndarray): Stress kernel for all patches and all evaluation points [m]. The first third of rows corresponds to the \( xx \) component, the second to \( yy \), and the third to \( xy \).
Notes

To compute the stress at the evaluation points given the shear and normal displacements \( D_s \) and \( D_n \) in each patch [m], with positive values meaning left-lateral shearing and crack opening, respectively, perform a matrix computation of the stress kernel with the displacements.

Example

Calculate the stress kernel for three observation points and two fault patches:

>>> import numpy as np
>>> x1, x2 = np.meshgrid([-1, 0, 1], [0])
>>> x1, x2 = x1.ravel(), x2.ravel()
>>> x1midfp, x2midfp = [0.5, 1.5], [-0.5, -1.5]
>>> halflen, theta, nu, E = 1/np.sqrt(2), np.pi/4, 0.25, 1
>>> K = Klinedisp(x1, x2, x1midfp, x2midfp, halflen, theta, nu, E)

Calculate the stress given left-lateral shearing and crack opening in both patches:

>>> Ds, Dn = [1, 1], [0.1, 0.1]
>>> S = K @ np.concatenate([Ds, Dn])
>>> print(S.round(3))
[ 0.084    nan  0.119 -0.       nan  0.     0.       nan  0.   ]

(The nan values are where the fault patch breaks the surface.)
