  9def Glinedisp(x1, x2, x1midfp, x2midfp, halflen, theta, nu):
 10    """
 11    Compute the displacement Green's functions for a shearing and opening
 12    line crack in a halfspace. Adapted for Python from the ``LDdispHS``
 13    Matlab function from [davis17]_. References to "C&S" in the source code
 14    mean [crouchstarfield83]_.
 15
 16    Parameters
 17    ----------
 18    x1 : float, numpy.ndarray
 19        :math:`x_1` location of evaluation points [m]
 20    x2 : float, numpy.ndarray
 21        :math:`x_2` location of evaluation points [m]
 22    x1midfp : float, numpy.ndarray
 23        :math:`x_1` location of the midpoint of the fault patches [m]
 24    x2midfp : float, numpy.ndarray
 25        :math:`x_2` location of the midpoint of the fault patches [m]
 26    halflen : float, numpy.ndarray
 27        Half-length of each fault patch [m]
 28    theta : float, numpy.ndarray
 29        The angle [rad] between the fault patch orientation (not the fault
 30        normal) and the :math:`x_1` axis in the mathematically negative sense,
 31        i.e. positive clockwise.
 32    nu : float
 33        Poisson's ratio [-]
 34
 35    Returns
 36    -------
 37    G : numpy.ndarray
 38        Displacement kernel for all patches and all evaluation points [m].
 39        The first half of rows corresponds to the :math:`xx` component,
 40        the second one to :math:`yy`.
 41
 42    Notes
 43    -----
 44
 45    To compute the displacement at the evaluation points given the shear and normal
 46    displacements :math:`D_s` and :math:`D_n` in each patch [m], with positive
 47    values meaning left-lateral shearing and crack opening, respectively,
 48    perform a matrix computation of the Green's functions with the displacements.
 49
 50    Example
 51    -------
 52
 53    Calculate the stress kernel for three observation points and two fault
 54    patches:
 55
 56    >>> import numpy as np
 57    >>> x1, x2 = np.meshgrid([-1, 0, 1], [0])
 58    >>> x1, x2 = x1.ravel(), x2.ravel()
 59    >>> x1midfp, x2midfp = [0.5, 1.5], [-0.5, -1.5]
 60    >>> halflen, theta, nu = 1/np.sqrt(2), np.pi/4, 0.25
 61    >>> G = Glinedisp(x1, x2, x1midfp, x2midfp, halflen, theta, nu)
 62
 63    Calculate the displacement given left-lateral shearing and crack opening in
 64    both patches:
 65
 66    >>> Ds, Dn = [1, 1], [0.1, 0.1]
 67    >>> D = G @ np.concatenate([Ds, Dn])
 68    >>> print(D.round(3))
 69    [ 0.289    nan -0.133 -0.094    nan  0.441]
 70
 71    (The ``nan`` values are where the fault patch breaks the surface.)
 72
 73    References
 74    ----------
 75
 76    .. [davis17] Davis, T. (2017).
 77       *A new open source boundary element code and its application to geological*
 78       *deformation: Exploring stress concentrations around voids and the effects*
 79       *of 3D frictional distributions on fault surfaces* (M.Sc thesis. Aberdeen University)
 80
 81    .. [crouchstarfield83] Crouch, S. L., & Starfield, A. M. (1983).
 82       *Boundary element methods in solid mechanics: with applications in rock*
 83       *mechanics and geological engineering*. Allen & Unwin.
 84    """
 85    # format shapes
 86    x1 = np.asarray(x1).reshape(-1, 1)
 87    x2 = np.asarray(x2).reshape(-1, 1)
 88    x1midfp = np.asarray(x1midfp).reshape(1, -1)
 89    x2midfp = np.asarray(x2midfp).reshape(1, -1)
 90    halflen = np.asarray(halflen).reshape(1, -1)
 91    beta = np.asarray(-theta).reshape(1, -1)
 92
 93    # input check
 94    if np.any(x2 > 0) or np.any(x2midfp > 0):
 95        raise ValueError("x_2 locations need to be zero or less in a halfspace.")
 96
 97    # Define material constant used in calculating influence coefficients.
 98    con = 1 / (4*np.pi * (1 - float(nu)))
 99    sb, cb = np.sin(beta), np.cos(beta)
100    s2b, c2b = np.sin(2*beta), np.cos(2*beta)
101    s3b, c3b = np.sin(3*beta), np.cos(3*beta)
102
103    # Define array of local coordinates for the observation grid relative to
104    # the midpoint and orientation of the ith element.
105    # Refer to (Figure 5.6, C&S, p. 91) and eqs. 4.5.1 of C&S, p. 57.
106    XB = (x1 - x1midfp)*cb + (x2 - x2midfp)*sb
107    YB = -(x1 - x1midfp)*sb + (x2 - x2midfp)*cb
108
109    # Coordinates of the image dislocation (equation 7.4.6 C&S)
110    XBi = (x1 - x1midfp)*cb - (x2 + x2midfp)*sb
111    YBi = (x1 - x1midfp)*sb + (x2 + x2midfp)*cb
112
113    # Fix roundoff errors in Ybi and Yb from trig function problems
114    YB[np.abs(YB) < 1e-10] = 0
115    YBi[np.abs(YBi) < 1e-10] = 0
116
117    # Calculate derivatives of the function f(x,y), eq. 5.2.5 of C&S, p. 81.
118    # which are used to calculate the displacement and stress components.
119    # It is understood that X and Y refer to XB and YB.
120    # First abbreviate repeated terms in the derivatives of f(x,y):
121    Y2 = YB**2
122    XMa, XPa = XB - halflen, XB + halflen
123    XMa2, XPa2 = XMa**2, XPa**2
124    R1S = XMa2 + Y2
125    R2S = XPa2 + Y2
126
127    # Same thing for the image dislocation
128    Y2i = YBi**2
129    XMai, XPai = XBi - halflen, XBi + halflen
130    XMa2i, XPa2i = XMai**2, XPai**2
131    R1Si = XMa2i + Y2i
132    R2Si = XPa2i + Y2i
133
134    # don't solve for locations where R1S(i) or R2S(i) are zero
135    skip_ix = (R1S == 0) | (R2S == 0) | (R1Si == 0) | (R2Si == 0)
136    R1S[skip_ix] = np.NaN
137    R2S[skip_ix] = np.NaN
138    R1Si[skip_ix] = np.NaN
139    R2Si[skip_ix] = np.NaN
140
141    R1S2i = R1Si**2
142    R2S2i = R2Si**2
143
144    # The following derivatives are eqs. 4.5.5a thru d of C&S, p. 58.
145    FF2 = con * (np.log(np.sqrt(R1S)) - np.log(np.sqrt(R2S)))
146
147    # get FF3 directly from engle differences without subindexing
148    FF3 = -con * np.angle(np.exp(1j*np.arctan2(YB, XMa))
149                          * np.exp(-1j*np.arctan2(YB, XPa)))
150
151    # get the next derivatives
152    FF4 = con * (YB / R1S - YB / R2S)
153    FF5 = con * (XMa / R1S - XPa / R2S)
154
155    # Calculate intermediate functions Fni for the image dislocation
156    FF2i = con * (np.log(np.sqrt(R1Si)) - np.log(np.sqrt(R2Si)))  # Equations 4.5.5 C&S
157    FF3i = -con * np.angle(np.exp(1j*np.arctan2(YBi, XMai))
158                           * np.exp(-1j*np.arctan2(YBi, XPai)))
159    FF4i = con * (YBi / R1Si - YBi / R2Si)
160    FF5i = con * (XMai / R1Si - XPai / R2Si)
161
162    # The halfspace examples of eqs. 5.5.3a and b of C&S, p. 91.
163    # See Appendix A of: Martel, S.J. and Langley, J.S., 2006. Propagation of
164    # normal faults to the surface in basalt, Koae fault system, Hawaii.
165    # Journal of Structural Geology, 28(12), pp.2123-2143.
166    FF6i = con * ((XMa2i - Y2i) / R1S2i - (XPa2i - Y2i) / R2S2i)
167    FF7i = 2*con * YBi * (XMai / R1S2i - XPai / R2S2i)
168
169    # Define material constants used in calculating displacements.
170    pr1 = 1 - 2*nu
171    pr2 = 2 * (1 - nu)
172    pr3 = 3 - 4*nu
173
174    # Calculate the displacement components using eqs. 5.5.4 of C&S, p. 91.
175    Gxx_s = -pr1*sb*FF2 + pr2*cb*FF3 + YB * (sb*FF4 - cb*FF5)
176    Gxy_n = -pr1*cb*FF2 - pr2*sb*FF3 - YB * (cb*FF4 + sb*FF5)
177    Gyx_s = +pr1*cb*FF2 + pr2*sb*FF3 - YB * (cb*FF4 + sb*FF5)
178    Gyy_n = -pr1*sb*FF2 + pr2*cb*FF3 - YB * (sb*FF4 - cb*FF5)
179
180    # See equations 7.4.8 and 7.4.9 in Crouch and Starfield
181    # Calculate IMAGE AND SUPPLEMENTAL DISPLACEMENT components due to unit SHEAR
182    # displacement discontinuity
183    Gxxi_s = (pr1 * sb * FF2i - pr2 * cb * FF3i +
184              (pr3 * (x2 * s2b - YB * sb) + 2 * x2 * s2b) * FF4i +
185              (pr3 * (x2 * c2b - YB * cb) - x2 * (1-2 * c2b)) * FF5i +
186              2 * x2 * (x2 * s3b - YB * s2b) * FF6i -
187              2 * x2 * (x2 * c3b - YB * c2b) * FF7i)
188    Gyxi_s = (-pr1 * cb * FF2i - pr2 * sb * FF3i -
189              (pr3 * (x2 * c2b - YB * cb) + x2 * (1-2 * c2b)) * FF4i +
190              (pr3 * (x2 * s2b - YB * sb) - 2 * x2 * s2b) * FF5i +
191              2 * x2 * (x2 * c3b - YB * c2b) * FF6i +
192              2 * x2 * (x2 * s3b - YB * s2b) * FF7i)
193
194    # Calculate IMAGE AND SUPPLEMENTAL DISPLACEMENT components due to unit NORMAL
195    # displacement discontinuity
196    Gxyi_n = (pr1 * cb * FF2i + pr2 * sb * FF3i -
197              (pr3 * (x2 * c2b - YB * cb) - x2) * FF4i +
198              pr3 * (x2 * s2b - YB * sb) * FF5i -
199              2 * x2 * (x2 * c3b - YB * c2b) * FF6i -
200              2 * x2 * (x2 * s3b - YB * s2b) * FF7i)
201    Gyyi_n = (pr1 * sb * FF2i - pr2 * cb * FF3i -
202              pr3 * (x2 * s2b - YB * sb) * FF4i -
203              (pr3 * (x2 * c2b - YB * cb) + x2) * FF5i +
204              2 * x2 * (x2 * s3b - YB * s2b) * FF6i -
205              2 * x2 * (x2 * c3b - YB * c2b) * FF7i)
206
207    Gxx = Gxx_s + Gxxi_s
208    Gxy = -(Gxy_n + Gxyi_n)
209    Gyx = Gyx_s + Gyxi_s
210    Gyy = -(Gyy_n + Gyyi_n)
211
212    # return stacked displacement kernel
213    Gx = np.hstack([Gxx, Gxy])
214    Gy = np.hstack([Gyx, Gyy])
215    G = np.vstack([Gx, Gy])
216    return G

219def Klinedisp(x1, x2, x1midfp, x2midfp, halflen, theta, nu, E):
220    """
221    Compute the stress kernel for a shearing and opening
222    line crack in a halfspace. Adapted for Python from the ``LDstressHS``
223    Matlab function from [davis17]_. References to "C&S" in the source code
224    mean [crouchstarfield83]_.
225
226    Parameters
227    ----------
228    x1 : float, numpy.ndarray
229        :math:`x_1` location of evaluation points [m]
230    x2 : float, numpy.ndarray
231        :math:`x_2` location of evaluation points [m]
232    x1midfp : float, numpy.ndarray
233        :math:`x_1` location of the midpoint of the fault patches [m]
234    x2midfp : float, numpy.ndarray
235        :math:`x_2` location of the midpoint of the fault patches [m]
236    halflen : float, numpy.ndarray
237        Half-length of each fault patch [m]
238    theta : float, numpy.ndarray
239        The angle [rad] between the fault patch orientation (not the fault
240        normal) and the :math:`x_1` axis in the mathematically negative sense,
241        i.e. positive clockwise.
242    nu : float
243        Poisson's ratio [-]
244    E : float
245        Young's modulus [Pa]
246
247    Returns
248    -------
249    K : numpy.ndarray
250        Stress kernel for all patches and all evaluation points [m].
251        The first third of rows corresponds to the :math:`xx` component,
252        the second to :math:`yy`, and the third to :math:`xy`.
253
254    Notes
255    -----
256
257    To compute the stress at the evaluation points given the shear and normal
258    displacements :math:`D_s` and :math:`D_n` in each patch [m], with positive
259    values meaning left-lateral shearing and crack opening, respectively,
260    perform a matrix computation of the stress kernel with the displacements.
261
262    Example
263    -------
264
265    Calculate the stress kernel for three observation points and two fault
266    patches:
267
268    >>> import numpy as np
269    >>> x1, x2 = np.meshgrid([-1, 0, 1], [0])
270    >>> x1, x2 = x1.ravel(), x2.ravel()
271    >>> x1midfp, x2midfp = [0.5, 1.5], [-0.5, -1.5]
272    >>> halflen, theta, nu, E = 1/np.sqrt(2), np.pi/4, 0.25, 1
273    >>> K = Klinedisp(x1, x2, x1midfp, x2midfp, halflen, theta, nu, E)
274
275    Calculate the stress given left-lateral shearing and crack opening in
276    both patches:
277
278    >>> Ds, Dn = [1, 1], [0.1, 0.1]
279    >>> S = K @ np.concatenate([Ds, Dn])
280    >>> print(S.round(3))
281    [ 0.084    nan  0.119 -0.       nan  0.     0.       nan  0.   ]
282
283    (The ``nan`` values are where the fault patch breaks the surface.)
284
285    """
286    # format shapes
287    x1 = np.asarray(x1).reshape(-1, 1)
288    x2 = np.asarray(x2).reshape(-1, 1)
289    x1midfp = np.asarray(x1midfp).reshape(1, -1)
290    x2midfp = np.asarray(x2midfp).reshape(1, -1)
291    halflen = np.asarray(halflen).reshape(1, -1)
292    beta = np.asarray(-theta).reshape(1, -1)
293
294    # input check
295    if np.any(x2 > 0) or np.any(x2midfp > 0):
296        raise ValueError("x_2 locations need to be zero or less in a halfspace.")
297
298    # The shear modulus, sm, is related to the prescribed elastic constants.
299    sm = E / (2 * (1 + float(nu)))
300    # Define material constant used in calculating influence coefficients.
301    con = 1 / (4*np.pi * (1 - float(nu)))
302    cons = 2 * sm
303    sb, cb = np.sin(beta), np.cos(beta)
304    s2b, c2b = np.sin(2*beta), np.cos(2*beta)
305    s3b, c3b = np.sin(3*beta), np.cos(3*beta)
306    s4b, c4b = np.sin(4*beta), np.cos(4*beta)
307
308    # Define array of local coordinates for the observation grid relative to
309    # the midpoint and orientation of the ith element.
310    # Refer to (Figure 5.6, C&S, p. 91) and eqs. 4.5.1 of C&S, p. 57.
311    XB = (x1 - x1midfp)*cb + (x2 - x2midfp)*sb
312    YB = -(x1 - x1midfp)*sb + (x2 - x2midfp)*cb
313
314    # Coordinates of the image dislocation (equation 7.4.6 C&S)
315    XBi = (x1 - x1midfp)*cb - (x2 + x2midfp)*sb
316    YBi = (x1 - x1midfp)*sb + (x2 + x2midfp)*cb
317
318    # Fix roundoff errors in Ybi and Yb from trig function problems
319    YB[np.abs(YB) < 1e-10] = 0
320    YBi[np.abs(YBi) < 1e-10] = 0
321
322    # Calculate derivatives of the function f(x,y), eq. 5.2.5 of C&S, p. 81.
323    # which are used to calculate the displacement and stress components.
324    # It is understood that X and Y refer to XB and YB.
325    # First abbreviate repeated terms in the derivatives of f(x,y):
326    Y2 = YB**2
327    XMa, XPa = XB - halflen, XB + halflen
328    XMa2, XPa2 = XMa**2, XPa**2
329    R1S = XMa2 + Y2
330    R2S = XPa2 + Y2
331
332    # Same thing for the image dislocation
333    Y2i = YBi**2
334    XMai, XPai = XBi - halflen, XBi + halflen
335    XMa2i, XPa2i = XMai**2, XPai**2
336    R1Si = XMa2i + Y2i
337    R2Si = XPa2i + Y2i
338
339    # don't solve for locations where R1S(i) or R2S(i) are zero
340    skip_ix = (R1S == 0) | (R2S == 0) | (R1Si == 0) | (R2Si == 0)
341    R1S[skip_ix] = np.NaN
342    R2S[skip_ix] = np.NaN
343    R1Si[skip_ix] = np.NaN
344    R2Si[skip_ix] = np.NaN
345
346    R1S2, R2S2 = R1S**2, R2S**2
347    R1S2i, R2S2i = R1Si**2, R2Si**2
348
349    FF4 = con * (YB / R1S - YB / R2S)
350    FF5 = con * (XMa / R1S - XPa / R2S)
351
352    # The following derivatives are eqs. 5.5.3a and b of C&S, p. 91.
353    FF6 = con * ((XMa2 - Y2) / R1S2 - (XPa2 - Y2) / R2S2)
354    FF7 = 2*con * YB * (XMa / R1S2 - XPa / R2S2)
355
356    FF4i = con * (YBi / R1Si - YBi / R2Si)
357    FF5i = con * (XMai / R1Si - XPai / R2Si)
358
359    # The halfspace examples of eqs. 5.5.3a and b of C&S, p. 91.
360    # See Appendix A of: Martel, S.J. and Langley, J.S., 2006. Propagation of
361    # normal faults to the surface in basalt, Koae fault system, Hawaii.
362    # Journal of Structural Geology, 28(12), pp.2123-2143.
363    FF6i = con * ((XMa2i - Y2i) / R1S2i - (XPa2i - Y2i) / R2S2i)
364    FF7i = 2*con * YBi * (XMai / R1S2i - XPai / R2S2i)
365
366    # don't estimate at invalid points
367    HR1 = ((halflen + XBi)**2 + YBi**2)
368    HR2 = (YBi**2 + (halflen - XBi)**2)
369    skip_ix = (HR1 == 0) | (HR2 == 0)
370    HR1[skip_ix] = np.NaN
371    HR2[skip_ix] = np.NaN
372
373    # *Tim* I used MATLABs symbolic to find these not eq's A.3 and A.4 of Martel
374    # Used Eq.A.1 on variable FF7i (expanded).
375    FF8i = (YBi * (1 / HR1**2 - 1 / HR2**2
376                   + (2 * (halflen - XBi) * (2 * halflen - 2 * XBi)
377                      ) / HR2**3
378                   - (2 * (halflen + XBi) * (2 * halflen + 2 * XBi)
379                      ) / HR1**3)
380            ) / (2 * np.pi * (nu - 1))
381    FF9i = (((halflen - XBi) / HR2**2
382             + (halflen + XBi) / HR1**2) / (2 * np.pi * (nu - 1))
383            - (YBi * ((4 * YBi * (halflen + XBi)) / HR1**3
384                      + (4 * YBi * (halflen - XBi)) / HR2**3)
385               ) / (2 * np.pi * (nu - 1)))
386
387    # Calculate the stress components using eqs. 5.5.5 of C&S, p. 92.
388    Kxx_s = 2*(cb*cb)*FF4 + s2b*FF5 + YB * (c2b*FF6-s2b*FF7)
389    Kxx_n = -FF5 + YB * (s2b*FF6 + c2b*FF7)
390    Kyy_s = 2*(sb*sb)*FF4 - s2b*FF5 - YB * (c2b*FF6-s2b*FF7)
391    Kyy_n = -FF5 - YB * (s2b*FF6 + c2b*FF7)
392    Kxy_s = s2b*FF4 - c2b*FF5 + YB * (s2b*FF6+c2b*FF7)
393    Kxy_n = -YB * (c2b*FF6 - s2b*FF7)
394
395    #  Calculate IMAGE AND SUPPLEMENTAL STRESS components due to unit SHEAR and
396    #  NORMAL displacement discontinuity
397    Kxxi_s = (FF4i - 3 * (c2b * FF4i - s2b * FF5i) +
398              (2 * x2 * (cb - 3 * c3b) + 3 * YB * c2b) * FF6i +
399              (2 * x2 * (sb - 3 * s3b) + 3 * YB * s2b) * FF7i -
400              2 * x2 * (x2 * c4b - YB * c3b) * FF8i -
401              2 * x2 * (x2 * s4b - YB * s3b) * FF9i)
402
403    Kxxi_n = (FF5i + (2 * x2 * (sb - 2 * s3b) +
404              3 * YB * s2b) * FF6i - (2 * x2 * (cb - 2 * c3b) +
405              3 * YB * c2b) * FF7i - 2 * x2 * (x2 * s4b - YB * s3b) * FF8i +
406              2 * x2 * (x2 * c4b - YB * c3b) * FF9i)
407
408    Kyyi_s = (FF4i - (c2b * FF4i - s2b * FF5i) -
409              (4 * x2 * sb * s2b - YB * c2b) * FF6i +
410              (4 * x2 * sb * c2b + YB * s2b) * FF7i +
411              2 * x2 * (x2 * c4b - YB * c3b) * FF8i +
412              2 * x2 * (x2 * s4b - YB * s3b) * FF9i)
413
414    Kyyi_n = (FF5i - (2 * x2 * sb - YB * s2b) * FF6i +
415              (2 * x2 * cb - YB * c2b) * FF7i +
416              2 * x2 * (x2 * s4b - YB * s3b) * FF8i -
417              2 * x2 * (x2 * c4b - YB * c3b) * FF9i)
418
419    Kxyi_s = (s2b * FF4i + c2b * FF5i +
420              (2 * x2 * sb * (1 + 4 * c2b) - YB * s2b) * FF6i +
421              (2 * x2 * cb * (3 - 4 * c2b) + YB * c2b) * FF7i +
422              2 * x2 * (x2 * s4b - YB * s3b) * FF8i -
423              2 * x2 * (x2 * c4b - YB * c3b) * FF9i)
424
425    Kxyi_n = ((4 * x2 * sb * s2b + YB * c2b) * FF6i -
426              (4 * x2 * sb * c2b - YB * s2b) * FF7i -
427              2 * x2 * (x2 * c4b - YB * c3b) * FF8i -
428              2 * x2 * (x2 * s4b - YB * s3b) * FF9i)
429
430    Kxx = np.hstack([Kxx_s + Kxxi_s, -Kxx_n - Kxxi_n])
431    Kyy = np.hstack([Kyy_s + Kyyi_s, -Kyy_n - Kyyi_n])
432    Kxy = np.hstack([Kxy_s + Kxyi_s, -Kxy_n - Kxyi_n])
433
434    # return stacked stress kernel
435    K = cons * np.vstack([Kxx, Kyy, Kxy])
436    return K

101    s2b, c2b = np.sin(2*beta), np.cos(2*beta)

108    YB = -(x1 - x1midfp)*sb + (x2 - x2midfp)*cb

111    XBi = (x1 - x1midfp)*cb - (x2 + x2midfp)*sb
112    YBi = (x1 - x1midfp)*sb + (x2 + x2midfp)*cb

115    YB[np.abs(YB) < 1e-10] = 0
116    YBi[np.abs(YBi) < 1e-10] = 0

118    # Calculate derivatives of the function f(x,y), eq. 5.2.5 of C&S, p. 81.
119    # which are used to calculate the displacement and stress components.
120    # It is understood that X and Y refer to XB and YB.
121    # First abbreviate repeated terms in the derivatives of f(x,y):

124    XMa2, XPa2 = XMa**2, XPa**2

126    R2S = XPa2 + Y2

129    Y2i = YBi**2

133    R2Si = XPa2i + Y2i

176    Gxx_s = -pr1*sb*FF2 + pr2*cb*FF3 + YB * (sb*FF4 - cb*FF5)
177    Gxy_n = -pr1*cb*FF2 - pr2*sb*FF3 - YB * (cb*FF4 + sb*FF5)

211    Gyy = -(Gyy_n + Gyyi_n)

214    Gx = np.hstack([Gxx, Gxy])
215    Gy = np.hstack([Gyx, Gyy])

420    Kxyi_s = (s2b * FF4i + c2b * FF5i +
421              (2 * x2 * sb * (1 + 4 * c2b) - YB * s2b) * FF6i +
422              (2 * x2 * cb * (3 - 4 * c2b) + YB * c2b) * FF7i +
423              2 * x2 * (x2 * s4b - YB * s3b) * FF8i -
424              2 * x2 * (x2 * c4b - YB * c3b) * FF9i)
425
426    Kxyi_n = ((4 * x2 * sb * s2b + YB * c2b) * FF6i -
427              (4 * x2 * sb * c2b - YB * s2b) * FF7i -
428              2 * x2 * (x2 * c4b - YB * c3b) * FF8i -
429              2 * x2 * (x2 * s4b - YB * s3b) * FF9i)
430
431    Kxx = np.hstack([Kxx_s + Kxxi_s, -Kxx_n - Kxxi_n])
432    Kyy = np.hstack([Kyy_s + Kyyi_s, -Kyy_n - Kyyi_n])
433    Kxy = np.hstack([Kxy_s + Kxyi_s, -Kxy_n - Kxyi_n])
434
435    # return stacked stress kernel
436    K = cons * np.vstack([Kxx, Kyy, Kxy])
437    return K

Compute the stress kernel for a shearing and opening line crack in a halfspace. Adapted for Python from the LDstressHS Matlab function from [davis17]_. References to "C&S" in the source code mean [crouchstarfield83]_.

Parameters
  • x1 (float, numpy.ndarray): \( x_1 \) location of evaluation points [m]
  • x2 (float, numpy.ndarray): \( x_2 \) location of evaluation points [m]
  • x1midfp (float, numpy.ndarray): \( x_1 \) location of the midpoint of the fault patches [m]
  • x2midfp (float, numpy.ndarray): \( x_2 \) location of the midpoint of the fault patches [m]
  • halflen (float, numpy.ndarray): Half-length of each fault patch [m]
  • theta (float, numpy.ndarray): The angle [rad] between the fault patch orientation (not the fault normal) and the \( x_1 \) axis in the mathematically negative sense, i.e. positive clockwise.
  • nu (float): Poisson's ratio [-]
  • E (float): Young's modulus [Pa]
Returns
  • K (numpy.ndarray): Stress kernel for all patches and all evaluation points [m]. The first third of rows corresponds to the \( xx \) component, the second to \( yy \), and the third to \( xy \).
Notes

To compute the stress at the evaluation points given the shear and normal displacements \( D_s \) and \( D_n \) in each patch [m], with positive values meaning left-lateral shearing and crack opening, respectively, perform a matrix computation of the stress kernel with the displacements.

Example

Calculate the stress kernel for three observation points and two fault patches:

>>> import numpy as np
>>> x1, x2 = np.meshgrid([-1, 0, 1], [0])
>>> x1, x2 = x1.ravel(), x2.ravel()
>>> x1midfp, x2midfp = [0.5, 1.5], [-0.5, -1.5]
>>> halflen, theta, nu, E = 1/np.sqrt(2), np.pi/4, 0.25, 1
>>> K = Klinedisp(x1, x2, x1midfp, x2midfp, halflen, theta, nu, E)

Calculate the stress given left-lateral shearing and crack opening in both patches:

>>> Ds, Dn = [1, 1], [0.1, 0.1]
>>> S = K @ np.concatenate([Ds, Dn])
>>> print(S.round(3))
[ 0.084    nan  0.119 -0.       nan  0.     0.       nan  0.   ]

(The nan values are where the fault patch breaks the surface.)
